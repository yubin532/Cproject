# 변수
> C언어에서 특정 하나의 데이터를 사용자가 이름 지어서 저장하는 방법(공간)입니다.<br>
> 변수에 저장되어 있는 데이터의 값은 일반적으로 언제든지 변경이 가능합니다.

## 변수를 만들 때 지켜줄 이름 규칙
1. 대소문자를 구분합니다.
2. 변수의 이름은 숫자가 맨 앞에 나올 수 없습니다.
3. 변수의 이름에 특수문자 사용이 불가능합니다
4. 변수의 이름으로 키워드(keyowrd)를 사용할 수 없습니다.

# 변수에 대한 선언과 초기화
1. 변수의 선언과 초기화를 동시에 진행할 수 있습니다.<br>
2. 데이터의 형태가 같다면, 여러 개를 한번에 선언할 수 있습니다. <br>

> ex)
printf("8진수 octo = %o\n", octo); <br>

printf("16진수 표기법 출력 hexa = %#x\n", hexa); <br>
printf("8진수 표기법 출력(8자리) octo : %0.8o\n", octo); <br>
printf("16진수 표기법 출력(8자리) hexa : %0.8x\n", hexa); <br>

> C언어에서의 변수는 각 정해진 데이터의 형태(자료형)이 존재한다.<br>

# 문자열 만드는 방법
char 변수명[문자의 개수]

1. C언어에서 한글을 입력하는 경우 한글 한 단어에 2칸을 소모합니다.
2. 현재 word는 10개의 단어를 사용할 수 있습니다.
3. 단 입력을 진행할 경우 문자의 개수보다 1개정도 적게 작성합니다.
4. 따라서 영단어는 9글자까지, 한글은 4글자까지 가능

> ex)

printf("단어를 입력해주세요 >> : ");
gets(word); //문자열 입력
puts(word); //전달할 문장 출력

# 연산자
프로그램에서 연산을 진행할 때 사용하는 기호 <br>
연산을 진행할 때 필요한 식이나 값을 항이라고 부릅니다.

> ex) 10 + 5
> > 연산자: + <br>
> > 10, 5라는 2개의 값이 필요하니 항이 2개<br>
1. 산술 연산자(기본적인 연산)<br>
+(더하기), -(빼기), *(곱하기) , /(나누기) , %(나머지)<br>
2. 대입 연산자<br>
   =를 이용해 값을 대입할수 있습니다.<br>
>> ex) a에 7을 대입하고 싶은 경우<br>
>> a = 7<br>
3. 혼합 대입 연산자<br>
= 전에 +,  -, * ,%를 작성하는 경우 대입하려는 값에 연산을 진행한 결과를 대입합니다.<br>
 =ㅁ를 이용해 값을 연산하여 대입할 수 있습니다.<br>
>> ex) a에 3을 더하고 이를 a에 대입하고 싶은 경우<br>
>> a =+ 3<br>

4. 증감 연산자<br>
   변수의 값을 1증가 / 감소 시킬 때 사용하는 연산자(단항)<br>
 **작성 위치에 따라 판정이 다르게 처리됩니다.**  <br>
++, --
>> ex) 후휘형 증감 연산: b = ++a <br>
>>연산 먼저 진행 후 a의 값을 1 증가 <br>

>> ex) 전위형 증감 연산: b = ++a <br>
>> a에 대한 1 증가 진행 후 연산을 처리 >> <br>

5. 관계 연산자<br>
연산자를 기준으로 왼쪽과 오른쪽 값의 크기를 비교하여 참 또는 거짓을 확인<br>
> 참일 경우: 1 / 거짓일 경우: 0
> if문 같이 진행

6. 교집합 a AND b<br>

7. 합집합 a OR b<br>

8. 합집합 - 교집합 a XOR b<br>
**XOR(Exclusive OR): 배타적 논리합, 대칭 차집합<br>
>> a와 b를 비교했을 때 비트가 다른 지점을 1로 같을 경우 0으로 처리<br>
이 연산 방식은 암호학의 기초로 많이 활용됩니다.
>> ex) 0101 -> 4 + 1 + 5

9. NOT 연산<br>
   비트를 반대로 처리하는 연산

10. shift 연산<br>
a << n a를 비트 n칸 이동한다.<br>
>> ex) a = 16;
c = a >> 4;
printf("c = %d\n", c);
**비트연산이란, 데이터를 비트 단위로 처리하는 연산자입니다.**
> 컴퓨터는 데이터를 2진코드로 관리합니다.(이진수)<br>
++ 2진수 계산법 ++
> 1. 계산할 값의 2의 제곱수로 분배.
> 2. 때 2의 제곱(1)도 포함.
> 3. 가장 큰 값부터 잘라낸다.

>> ex) 제곱수 1 2 4 8 16 32 64 128 ... <br>
11 = 8 +  ... + 2 + 1 --> 1011
비트 중에서 겹치는 부분만 1로 처리하며 나머지는 0으로 처리
